# Hegemony: Lead Your Class to Victory — Rules Reference

> **Source:** Hegemony English Rulebook v1.2 ([PDF](rules.pdf))
> **Publisher:** Hegemonic Project Limited
> **Game Design:** Varnavas Timotheou, Vangelis Bagiartakis
> **Players:** 2-4
> **Rounds:** 5

Hegemony is an asymmetric, card-driven board game in which players take the role of a specific group: the Working Class, the Middle Class, the Capitalist Class, or the State itself. Each player plays differently and tries to pursue their own agenda.

## Class Goals at a Glance

| Class                | Goal                                        | Primary VP Source                                           |
| -------------------- | ------------------------------------------- | ----------------------------------------------------------- |
| **Working Class**    | Increase Prosperity by covering needs       | Prosperity track, Trade Unions, Socialist Policies          |
| **Middle Class**     | Increase Prosperity while running Companies | Prosperity track, operational Companies, Section B Policies |
| **Capitalist Class** | Maximize profit                             | Capital/Wealth track, Neoliberal Policies, stored goods     |
| **State**            | Maintain Legitimacy across all Classes      | Legitimacy tracks, Political Agenda, Events                 |

## Round Structure

Each of the 5 rounds consists of:

1. **Preparation Phase** — Set up the new round (draw cards, get Workers, etc.)
2. **Action Phase** — 5 turns per player; play cards or perform Basic/Free Actions
3. **Production Phase** — Companies produce, Wages paid, needs covered, taxes paid
4. **Elections Phase** — Vote on proposed Bills; Influence can sway results
5. **Scoring Phase** — Score VP based on role-specific criteria
